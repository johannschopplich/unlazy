# @unlazy/core

[![npm version][npm-version-src]][npm-version-href]

Core engine for [unlazy](https://unlazy.byjohann.dev). **You probably don't want to install this directly.** Pick the entry point that matches your stack:

- **No framework**: [`unlazy`](https://www.npmjs.com/package/unlazy) (re-exports the full public API of `@unlazy/core`)
- **Vue**: [`@unlazy/vue`](https://www.npmjs.com/package/@unlazy/vue)
- **Nuxt**: [`@unlazy/nuxt`](https://www.npmjs.com/package/@unlazy/nuxt)
- **React**: [`@unlazy/react`](https://www.npmjs.com/package/@unlazy/react)
- **Solid**: [`@unlazy/solid`](https://www.npmjs.com/package/@unlazy/solid)
- **Svelte**: [`@unlazy/svelte`](https://www.npmjs.com/package/@unlazy/svelte)

The framework adapters depend on `@unlazy/core` internally. Install `@unlazy/core` directly only if you're building your own integration.
